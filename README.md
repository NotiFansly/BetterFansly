# BetterFansly

**A power-user client modification for Fansly**

Enhancing your experience with themes, utilities, and privacy tools.

---

## Features

### Themes
- **Catppuccin Theme** with 4 flavors (Latte, Frappé, Macchiato, Mocha) and 14 accent colors
- Custom CSS injection support
- Remote theme loading from URLs

### Core Plugins
- **Stream Miniplayer** - Draggable PiP player with quality selector and transparency mode
- **Mutual Indicator** - "Follows You" badges next to usernames
- **Keyword Muter** - Hide posts containing specific words or phrases
- **Spending Tracker** - Analyze your purchase history and see top creators
- **Backup & Migration Tools** - Export/import your following list between accounts

### Extensibility
- Install custom JavaScript plugins directly from the UI
- Full Chrome UserScripts API support for advanced modifications
- Plugin library system for community scripts

---

## Installation

1. Download the latest release or clone this repository
2. Open Chrome and navigate to `chrome://extensions/`
3. Enable **Developer Mode** (top right)
4. Click **Load Unpacked** and select the extension folder
5. Visit [Fansly](https://fansly.com) and look for the BetterFansly button in Settings or your profile dropdown

---

## Quick Start

1. Click the **BetterFansly** button in the sidebar or settings page
2. Navigate through tabs to enable plugins or customize themes
3. All settings are saved automatically to your browser

---

## Privacy & Safety

- **No data collection** - Everything runs locally in your browser
- **No external servers** - All API calls are direct to Fansly
- **Open source** - Audit the code yourself
- **Custom plugins** - Only install scripts from trusted sources

**Warning:** Installing untrusted custom plugins can compromise your account. Always review code before installation.
